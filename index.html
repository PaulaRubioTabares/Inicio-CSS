Iniciando en css

Hay varias formas de escribir css para una página, la más ordenada, es importado un archivo por aparte en donde esté todo el código css, aunque el código de css también se puede escribir dentro del archivo html.

En esta práctica el primer css aplicado es que a todo el texto que este dentro de la etiqueta body va a tener un color morado, pero con un selector que se va a ver más adelante pasara a ser negro

Pero si se quiere que un elemento en concreto sea diferente, entonces se utilizan los selectores: id y class, por ejemplo:

  • id="titulo", para el título de esta página
  • class="texto" para el tercer párrafo de la página

Para llamar un selector en las hojas de estilo es:

  • Para id #nombredelselector, ejemplo: #titulo
  • Para class .nombredelselector, ejemplo: .texto

Se debe tener cuidado con los nombres que se asignan a los selectores, por ejemplo:

En este título se tiene el selector class="texto"

En este párrafo se tiene el selector class="texto"

En este párrafo se tiene el selector class="texto especial"

En este párrafo se tiene el selector class="especial"

Se observa que con todos los selectores de tipo class que tienen la palabra texto, se aplica el estilo, no importa si tienen más texto

Si se desea que solo los párrafos tengan color de letra y no otras etiquetas, entonces en la hoja de estilos se debe escribir p.texto

En esta parte se observa lo antes comentado, se selecciona que los párrafos sean los que se van a modificar y no otras etiquetas, en este caso colocando en css p.letra

En este título se tiene el selector class="letra"

En este párrafo se tiene el selector class="letra"

En este párrafo se tiene el selector class="letra diferente"

En este párrafo se tiene el selector class="diferente"

Ahora sí solo las etiquetas de p que tengan el selector class y la palabra letra son los que se modificaron

En css se pueden sobreponer estilos, esto ya que el estilo que se aplique va a ser el último declarado, para este ejemplo en la hoja de estilos se va a colocar primero que:

p.palabra va a ser de color rojo y debajo de esa instrucción se coloca que: p.especifica va a ser de color verde

En este párrafo se tiene el selector class="palabra"

En este párrafo se tiene el selector class="palabra especifica"

En este párrafo se tiene el selector class="especifica"

Por ello el orden en que se escribe el código en css es crucial

En esta parte se va a colocar parte del contenido dentro de un div, para aplicar un estilo solo a los p que hay dentro de ese div

Para ello en css se coloca div p{}

Este párrafo está dentro del div

Este h2 está en el div

Este h3 también está dentro del div

Este es otro p que está en el div

En el caso de h2 y h3 se están seleccionando a la vez y solo los que están dentro del div : div h2, div h3 {}

El ultimo selector que se tiene es el selector universal: *, este selecciona todos los elementos a los que no se les ha definido su propiedad,
por ejemplo, con ese selector se va a colocar todo el texto del mismo tamaño, y se le va a decir que el color va a ser negro, pero si ya antes se ha designado un color a la letra, no lo va a modificar.
